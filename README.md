# MEGAcmd GUI

Web-based user interface for MegaCMD. Work in progress...


## Development

* Build image and start container:
  ```bash
  docker-compose up --build -d
  ```
* Start shell in container:
  ```bash
  docker exec -it megacmd-gui /bin/bash
  ```
